Tools
Info

User can click on + sign to add the blocks into canvas, User will have tools based on data configued by Admin


User can get extra special tool by adding Coupon Code. User can ask anyone to add mentioned coupon code into any jira issue and that jira issue's key can be use for validation. If found comment with coupon code , count of special tools will be added automatically.


User can share current canvas to other user via mail or any other sytem tool.


Double Click will set block into static fixed position


If user want to delete block , they can touch block to Delete Icon (Top Right - second icon)


If user want to change the color of block , they can touch block to Color Picker Icon (Top Right - First Icon)


Have Fun!
